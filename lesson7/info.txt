ls -> узнать содержимое директории
touch 1.tx -> создается файл 1.txt (найти аналог команды для виндовс)
mkdir назвагие_папки -> создает папку(директорию)
ls ../ -> вернуться в предыщию директорию
mv старое_название_файла_или_папки новое_название_папки_или_файла -> команда чтобы переименоваться
cp откуда_копировать куда_копировать
